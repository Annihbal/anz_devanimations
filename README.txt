# Animation Menu for REDM

This project contains an fxanimation menu for REDM, created with the VORP framework.

## Installation

1. Download the project.
2. Place the project folder in your RedM resource directory.
3. Add `ensure anz_devanimations` to your `server.cfg` file.

## Commands 

1. /devanime for opening the postfxanimation menu 
2. backspace key for closing it!
